Comparison operators 비교연산자

  == : 일반적인 같음(타입을 고려하지 않음)
  === : 타입까지 고려하여 엄격하게 같음
  != : 같잖음
  !== : 엄격하게 같지 않음
  ? = : 삼항 연산자
  >=, <= : 둘 중에 하나의 조건만 맞으면 true
  - true나 false

Logical Operators

  && logical and 두 조건이 성립해야지만 참
  || logical or 둘 중 하나만 성립해도 참
  ! logical not 같지 않으면 true
