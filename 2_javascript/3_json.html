Java Script Object Notation 

      - 데이터를 저장하고 전송하는 형식입니다. 
      - 서버에서 웹페이지로 데이터를 전송할 때 자주 사용됩니다. 
      - JSON은 데이터를 읽고 생성하는 모든 프로그래밍 언어로 작성할 수 있습니다.
      (php, asp, jsp)모두 사용이 가능합니다.
      - JSON.parse() : json문자열을 자바스크립트로 변환한다.
      - JSON.stringify() : 객체를 JSON문자열로 변환합니다. 

      JSON의 syntax rules
      - 데이터가 이름 / 값 쌍으로 되어 있습니다.
      {"name" : "young-il"}

      "key" : "value"
      - 데이터는 쉼표로 구분됩니다.
      - 중괄호는 객체를 고정합니다.
      - 대괄호는 배열을 보유합니다.
