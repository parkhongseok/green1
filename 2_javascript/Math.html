Constants 8

    Math.E : 오일러 상수
    Math.PI : 원주율
    Math.SQRT2 : 
    Math.SQRT1_2 : 1/2의 제곱근을 리턴
    Math.LN2 : 2의 자연로그를 리턴
    Math.LN10 : 10의 자연로그를 리턴
    Math.LOG2E : 2^e인 로그를 리턴
    Math.LOG10E : 10^e인 로그를 리턴

Number to Integer : 숫자를 정수로

  Math.round : 소수를 가까운 정수로 : 올림
  Math.ceil : 소수를 가까운 정수로 : 올림
  Math.floor : 소수를 가까운 정수로 : 내림
  Math.trunc : 정수 부분을 리턴

  Math.sign() : x가 음수, null 또는 양수이면 리턴
  Math.pow(8, 2) : 8^2
  Math.sqrt() : 제곱근을 리턴
