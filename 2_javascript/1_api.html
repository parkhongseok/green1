Geolocation(위치정보)

사용자의 지리적 위치를 얻는데 사용합니다
사용자의 승인이 없으면 해당 위치를 사용할수 없습니다
위치정보는 스마트폰과 같이 gps가 탑재된 기기에서 가자 정확합니다
- getCurrentPosition() : 사용자의 위치를 리턴하는데 사용합니다
- watchPosition() : 사용자의 현재 위치를 리턴하고 움직일때 마다 업데이트된
위치를 계속해서 리턴합니다.
- clearWatch() : watchPosition()메소드를 중지 시킵니다
